##  Скрипт  `seller.py`

Скрипт обновляет сведения о количестве товаров и их стоимости на Ozon, используя данные об остатках и ценах на складе, полученных с сайта [timeworld.ru](https://timeworld.ru/)

### Запуск

Запускается скрипт командой:

`python seller.py`
